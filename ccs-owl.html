OWL Document Reader

An enterprise-grade document processing system leveraging AWS Textract OCR to transform traditional financial workflows into streamlined digital operations with high accuracy and efficiency.

Features

  • Fully AWS integrated and serverless
  • Mobile-friendly UI to scan and upload documents on the go
  • Textract-supported flexible document recognition, without need for templates

Core Technologies

  • AWS Textract OCR with dynamic confidence thresholds
  • TypeScript-based frontend hosted on AWS Amplify
  • Python backend with automated document validation workflows
  • Amazon S3 for document blob storage
  • Amazon RDS for structured metadata and indexing
  • RESTful API integration between frontend and backend
